### AtlasP Backend

To run locally run:
    yarn
    yarn dev
    Head to: [localhost:3001](https://localhost:3001) to verify the server is up
    Request to localhost:3001!

To run prouduction:
    Turn on docker-machine
    cd into root folder (/atlasP-backend/)
    docker-compose build && docker-compose up
